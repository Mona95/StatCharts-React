# Sample Project - Charts

#### Used Dependencies

- @material-ui/core
- axios
- chart.js
- react-chartjs-2
<hr/>

#### How to run project

- Unzip the folder.
- run `npm install`.
- run `npm start`.

<hr/>

#### Folder structure

```
components
        └─── BarChart
        └─── LineChart
containers
        └─── App
        └─── Charts

```
